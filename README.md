# VGM Screenshot Embedder

A Python CLI tool to embed VGM screenshots into files.

## Usage

### Quick Start with `uvx`

Prerequisites: [uvx](https://docs.astral.sh/uv/getting-started/installation/)

Run directly from GitHub without installing:

```shell
uvx --from git+https://github.com/jmaroeder/vgm-screenshot-embedder.git vgm-screenshot-embedder
```

### Installation

```shell
uv tool install --from git+https://github.com/jmaroeder/vgm-screenshot-embedder.git vgm-screenshot-embedder
```

## Development

Clone and install from the repository:

```bash
git clone https://github.com/jmaroeder/vgm-screenshot-embedder.git
cd vgm-screenshot-embedder
uv sync
```

## Development
